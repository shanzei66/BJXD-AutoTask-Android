# 北京现代自动任务 APP (安卓端)
北京现代自动任务 APP 的安卓端源码。

## 功能
- 手机号短信登录后，点击添加账号，自动获取 token 保存到账号列表
- 支持多个账号，支持账号排序，删除
- 支持配置 腾讯混元大模型 API Key，实现 AI 自动答题
- 支持手动答题，应用会通过弹窗方式显示当前题目跟答案
